    # 🕵️ Talent Shadow Hunter | 影子人才挖掘器

在小红书等社交平台通过**评论行为分析**而非**内容发布**来识别高价值隐藏人才（特别是量化、Web3、AI领域）。

## 🎯 核心洞察

**发布者悖论**：90%的行业大佬不生产内容，但无法抑制评论的冲动。
- 发帖子 = 暴露身份 + 引来打扰
- 写评论 = 满足表达欲 + 保持匿名性

## 🔍 应用场景

| 目标人群 | 识别信号 | 挖掘需求 |
|---------|---------|---------|
| 量化研究员 | 在因子挖掘帖下讨论细节 | 跳槽/募资/创业 |
| 基金经理 | 评论宏观分析但主页空白 | 寻找新的平台 |
| Web3开发者 | 技术贴下深度纠错 | 寻找合伙人 |
| AI研究员 | 讨论论文实现细节 | 看机会 |
